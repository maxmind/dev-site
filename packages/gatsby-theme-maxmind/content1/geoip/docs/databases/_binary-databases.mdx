Binary databases make use of the [MaxMind DB file format](https://maxmind.github.io/MaxMind-DB/).
MaxMind provides official API clients in the language listed below. The
following API clients are open source and licensed under the Apache License,
Version 2.0.

You can also use the [mmdbinspect tool](https://github.com/maxmind/mmdbinspect)
(in beta), a command line interface built with Go, to look up one or more IPs
from one or more MMDB databases and receive output in a parsable JSON format.
